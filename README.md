# 🐱 Cat Cafe - Café & Compañía Felina

## 📖 Descripción

**Cat Cafe** es una landing page moderna y completamente responsive para una cafetería temática de gatos. El sitio web combina un diseño cálido y acogedor con funcionalidades interactivas para mostrar el menú de cafés especiales y presentar a los gatos residentes del establecimiento.

### ✨ Características Principales

- 🎨 **Diseño moderno y cálido** - Paleta de colores inspirada en tonos café y beige
- 📱 **Completamente responsive** - Adaptable a móviles, tablets y desktop
- ☕ **Catálogo de cafés** - 6 productos con imágenes, descripciones y precios
- 🐈 **Carrusel interactivo de gatos** - 8 gatos con desplazamiento suave y controles manuales
- 🧭 **Navegación sticky** - Barra de navegación fija con efecto blur
- 🔍 **Smooth scrolling** - Desplazamiento suave entre secciones
- 🎯 **Lazy loading** - Carga optimizada de imágenes
- 📞 **Contacto interactivo** - Modal de contacto con información del negocio

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
|------------|-----|
| HTML5 | Estructura semántica |
| CSS3 | Flexbox, Grid, variables CSS, backdrop-filter |
| JavaScript (ES6) | Carrusel dinámico, eventos, comportamiento responsive |
| Font Awesome 6 | Íconos vectoriales |
| Google Fonts | Fuente 'Inter' |

## 🖥️ Instalación y Uso Local

### Opción 1 - Directa
1. Descarga el archivo `index.html`
2. Haz doble clic para abrirlo en tu navegador

### Opción 2 - Con servidor local
```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .

# VS Code Live Server
# Instala la extensión y haz clic en "Go Live"
```
